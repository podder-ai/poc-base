# poc-base

This is base repository for PoC (Proof of Concept) code.

## Usage

### Preparation

```bash
$ git clone git@github.com:takp/poc-base.git
$ cd poc-base
$ pip install -r requirements.txt
$ cp .env.sample .env
```

Add your env var to `.env` file.

### Add your code

Add your code to `app/task.py`.

### Run

```bash
$ python main.py
```
